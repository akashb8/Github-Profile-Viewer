# 🔍 GitHub Profile Viewer

A simple and interactive web application that allows users to search and view GitHub profiles in real-time using the GitHub API.

---

## 🚀 Features

- 🔎 Search any GitHub user by username  
- 👤 View profile details:
  - Avatar  
  - Name  
  - Bio  
  - Location  
  - Public repositories count  
- 📂 Display user repositories  
- ⚡ Fast and responsive UI  
- ❌ Handles invalid usernames gracefully  

---

## 🛠️ Tech Stack

- **Frontend:** React.js / HTML / CSS / JavaScript  
- **API:** GitHub REST API  
- **Styling:** Tailwind CSS / Bootstrap / CSS
